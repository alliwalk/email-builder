In one project, I redesigned the email marketing design tools to help users build attractive email templates without technical or design knowledge.

Problem

Clumsy tools led to customer designs that lacked nuance and hurt the Infogroup brand.

Strategy

Fix an email marketing tool that didn’t do its best for customers. The idea was to create tools that didn’t require design skill or technical knowledge.

In my design explorations, I borrowed design patterns from competitors, especially if they included a mailing list.

Actions

The workflow and the framework of the email builder were rebuilt to progressively add to an email template. I created an updated template gallery and new UI elements, to help users easily create an email template.

Setting Up Your Email Template

1. Template Gallery Users start their email design by selecting a template from the gallery. Templates are organized by industry and type.

2. Customize layout In this step, the layout could be further refined. Users customize layout by rearranging or removing content modules, using a drag-and-drop user interaction.

3. Select color theme Next add color. Users select a color theme in coordinated palettes.

4. Add content To customize content, simply click and edit with context menus.
