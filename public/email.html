Email Marketing Builder

In one project, I redesigned tools used by businesses to build email templates for marketing campaigns.

Problem

The old tools were too complicated for design amateurs. They didn’t serve customers and hurt the Infogroup brand.

Strategy

In order to create design tools that didn't require design knowledge, I researched competitors to find common design patterns.

Actions

The workflow and the framework of the email builder were rebuilt, including an updated template gallery and new UI elements. Users could now design their email template in a logical and sequential order, to help them build an attractive marketing campaign.

Outcome

The product was released as the Email Campaign Builder, available for both InfoUSA and Salesgenie customers.

Setting Up Your Email Template

High-Fidelity Wireframes

1. Select template Users start their email design by selecting a template from the gallery. Templates are organized by industry and type.

2. Customize layout In this step, the layout could be further refined. Users customize layout by rearranging or removing content modules, using a drag-and-drop user interaction.

3. Select color theme Next add color. Users select a color theme in coordinated palettes.

4. Add content To customize content, simply click and edit with context menus.
